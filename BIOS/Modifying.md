### How to change some information in BIOS
Using DMI Edit via Windows Shell for AMI BIOS

in Command Prompt run the following commands
for changing System Manufacture
AMIDEWINx64.EXE /SM ["My Manufacture"]
for changing System Product 
AMIDEWINx64 /SP ["My Product"]

name of application DMI-edit-win64-ami.zip
download from : download.schenker-tech.de/package/dmi-edit-efi-ami/
